repmat {multivariateAnomaly}R Documentation

Replicate matrix.

Description

Analogous to the repmat function in matlab. A = repmat(B, m, n) creates a new matrix A having m * n tiling of B. The size of the matrix A will be [nrow(B) * m, ncol(B) * n]

Usage

repmat(B, m, n)

Arguments

B The input matrix that has to be replicated.
m Number of tilings along the row.
n Number of tilings along the column.

Author(s)

Sriram Lakshminarasimhan

Examples

B <- matrix(c(1:9), nrow=3)
B
A <- repmat(B, 2, 3)
A
